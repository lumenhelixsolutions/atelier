Install in seconds

Run atelier locally by installing the FastAPI backend and React + Vite UI side by side.

cd api && python3 -m venv ../.venv && ../.venv/bin/pip install -r requirements.txt
cd ../app && npm install && npm run dev
cd api
python -m venv ..\.venv
..\.venv\Scripts\pip install -r requirements.txt
cd ..\app
npm install
npm run dev
cd api && python3 -m venv ../.venv && ../.venv/bin/pip install -r requirements.txt
cd ../app && npm install && npm run dev

Tested on Windows 11, macOS Sonoma, Ubuntu 22.04/24.04, and modern mobile browsers.

How it works

A two-layer shell: FastAPI host mounts agent routes; React UI supervises formation.

atelier/
├── api/      FastAPI host — mounts DropClass routes
├── app/      React (Vite) UI — /formation, supervisor dashboard
└── scripts/  PowerShell starters
